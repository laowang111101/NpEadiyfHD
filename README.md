# 网络小说数据分析系统 python195

## 项目概述

本项目是一个基于Python的网络小说数据分析系统，采用了Hive、Flask、爬虫（Spider）、MySQL和Vue等技术与工具，实现了对网络小说数据的采集、存储、分析和展示。

## 技术栈

- **后端**: Python 3.7
- **Web框架**: Flask
- **大数据处理**: Hive
- **数据库**: MySQL 5.7
- **前端**: Vue
- **数据采集**: 爬虫

## 功能模块

- 数据采集：自动抓取网络小说内容
- 数据存储：将采集的数据存储至Hive和MySQL
- 数据分析：对存储的小说数据进行多维度分析
- 数据展示：通过Web界面展示分析结果

## 系统角色

- 数据采集器：负责定期抓取网络小说数据
- 数据处理器：负责清洗和转换采集到的数据
- 数据库：负责存储处理后的数据
- 分析引擎：对存储的数据进行统计分析
- Web应用：提供用户界面，展示分析结果

## 运行环境

- Python 3.7+
- Flask
- Hive
- MySQL 5.7
- Vue

## 部署步骤

1. 安装Python 3.7+
2. 安装并配置Hive
3. 安装并配置MySQL 5.7
4. 部署Flask后端服务
5. 部署Vue前端应用
